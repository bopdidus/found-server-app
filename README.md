# Purpose

this project is the server side of an application, where people can find lost things. The aim of the application is, to allow people to post lost things, in such a way that the owner refind his things.

## Technologies

I use the framework nodejs with: 
* his library express 
* typeorm for the management of database.
* Jwt for the authentication

# Awesome Project Build with TypeORM

Steps to run this project:

1. Run `npm i` command
2. Setup database settings inside `ormconfig.json` file
3. Run `npm start` command

> I use Github actions for CI (continuous Intergration) and CD (Continuous Delivery)
